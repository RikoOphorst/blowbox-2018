Detailed Description

A "malloc" allocator, uses regular malloc/free.

This allocator guarantees that the underlying memory allocation is done properly. If any issues arise with the memory interface, they can be found with this specific allocator. As long as memory is not a bottleneck, this memory allocator should be used.
