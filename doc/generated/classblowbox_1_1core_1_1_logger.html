Detailed Description

Allows you to log Messages.

The Logger allows you to log Messages from anywhere inside of the entire blowbox project. By default, logged messages are not outputted anywhere. If you want to catch Messages being logged, you must define the Logger::message_receiver struct. This function is called with a custom userdata void* which is settable by setting Logger::message_receiver_userdata.
